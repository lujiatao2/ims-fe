# IMS——Inventory Management System（前端）

&emsp;&emsp;库存管理系统，用于Web/接口自动化测试的实战项目，包含了登录、注销、入库、出库、物品管理、物品类别管理等功能。

## 技术栈

Java  
Spring/Spring Boot/Spring Cloud  
MyBatis  
HTML/CSS/JavaScript  
Vue.js  
Element
